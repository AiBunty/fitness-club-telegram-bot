# QUICK_REFERENCE – Cheat Sheet

## Telegram Commands
- /start – Register or open menu
- /menu – Main menu
- /checkin – Daily attendance
- /shake – Request shake
- /stats – View stats
- /admin – Admin panel
- /staff – Staff dashboard

## SQL Queries
- See documentation for common queries

## Database Management
- Backup: `pg_dump fitness_club_db > backup.sql`
- Restore: `psql fitness_club_db < backup.sql`

## Troubleshooting
- Check logs: `logs/fitness_bot.log`
- Test DB: `python test_db.py`
- Restart bot if unresponsive
