# Fitness Club Bot – How to Use (Admin & User)

## User Quick Start
- Send `/start` to register (name, phone, age, weight, optional photo).
- Open `/menu` anytime to log or view progress.
- Fast logging commands:
  - `/weight` – update weight
  - `/water` – log water intake
  - `/meal` – snap meal photo
  - `/checkin` – record gym check-in
  - `/habits` – daily habit confirmation
- Progress & rewards:
  - View points, badges, and leaderboard from the menu.
  - Join challenges and track progress inside the menu.
- Payments & shakes:
  - Use menu options to see payment status or order shakes when enabled.

## Admin Quick Start
- Approvals:
  - `/pending_attendance` – review and approve attendance check-ins.
  - `/pending_shakes` – review shake orders.
- Payments & challenges:
  - `/payment_status` – monitor member payment status.
  - `/challenges` – view active challenges; `/my_challenges` for personal view.
- Analytics:
  - `/admin_dashboard` – key metrics and trends.
- Notifications:
  - `/notifications` – view and manage alerts.

## Tips for Everyone
- After registration, `/menu` is the fastest way to navigate.
- Use the quick commands for routine logging to save time.
- Keep contact details current so admins can reach you.
- If something feels stuck, send `/cancel` to exit any flow and start again.

## Support
- Still need help? Share a screenshot of the error/message with an admin so they can follow up quickly.
